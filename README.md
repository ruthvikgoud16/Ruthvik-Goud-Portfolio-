# Ruthvik Goud | Interactive AI & Hardware Engineering Workbench

Welcome to my custom engineering portfolio and interactive systems laboratory. Designed and built from the ground up, this system models the intersection of **Electrical & Electronics Engineering (ECE)** and **Neural Agent Automation**.

## 🚀 Live Environment Specs
- **Production Build:** Optimized for Vercel & modern serverless runners
- **Frameworks:** React 19 + TypeScript + Vite + Tailwind CSS
- **Motion Engine:** Motion (Framer Motion v12) for 3D perspective viewport transforms
- **API Runtime:** Express CJS bundle for robust server endpoints

---

## 🛠️ Integrated Lab Portals

### 1. **Pilot Bio & Mission Brief** (`/src/components/About.tsx`)
A 3D holographic workspace perspective card that tilts dynamically in response to your cursor coordinates. It houses my personal dossier, education details as an ECE student at **Sreenidhi Institute of Science and Technology (SNIST)**, and my professional support work on the Google Process at **Concentrix**.

### 2. **Interactive Hardware Command Center** (`/src/components/EngineeringLab.tsx`)
A tactile FR4 PCB motherboard model constructed in custom vector mathematics and SVG. It models core hardware nodes (Neural Processor, Arduino Controller, Comms Transceivers, Server database arrays) with interactive diagnostics:
- Custom SVG bezier circuit lines that stream animated current packet pulses.
- Real-time diagnostic logs feeding data schemas and technologies.
- Fully responsive fallback lists for mobile/tablet screens.

### 3. **Featured Engineering Runs** (`/src/components/Projects.tsx`)
Interactive cards featuring systems designed under pressure during national-scale code runs:
- **CrisisOS Command** (HackPrix Season 2 First Place Winner)
- **Redrob Truth Engine** (India.Runs Winner)
- **AutoOps Enterprise Hub** (NVIDIA Hackathons)
- **PID Self-Leveling Drone Module** (Makers Arena Coimbatore)

### 4. **Adaptive Companion Terminal** (`/src/components/AICompanionChatbot.tsx`)
A holographic chatbot powered by the Gemini 3.5 Flash serverless adapter. It features custom ECE-influenced personality buffers, contextual viewport section awareness, and reliable client-side cache fallbacks so it stays online regardless of your hosting configuration.

## 🏎️ Local Installation & Development

To power up the workbench locally, run:

```bash
# Clone the system and navigate in
git clone https://github.com/ruthvikgoud16/portfolio-system.git
cd portfolio-system

# Install workspace dependencies
npm install

# Start the dev server in TSX mode
npm run dev
```

The system will initialize a real-time hot-reloading development server forwarding traffic.

## 📦 Production Compiles & Vercel Deployment

This portfolio is fully optimized to be **Vercel deployment ready**:

### Client Static Compiles (SPA Mode)
Vercel's default build pipeline works beautifully with zero-configuration out-of-the-box.
- **Build Command:** `npm run build`
- **Output Directory:** `dist`

### Configuration (`vercel.json`)
For streamlined routing, a custom `vercel.json` is packaged at the root to rewrite client paths properly and handle API server routes seamlessly.
